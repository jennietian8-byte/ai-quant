# TASK4 AI Quant Lab Static Site

This directory is a pure static GitHub Pages site for the TASK4 turtle strategy assignment.

Pages:

- `index.html`: AI Quant Lab portal.
- `turtle.html`: Turtle strategy playground with adjustable parameters.
- `indicator.html`: indicator lab demo.
- `ma-cross.html`: moving-average crossover demo.
- `workflow.html`: GitHub Actions deployment explanation.

The site uses local CSV files in `data/` and does not require a backend, token, cookie, or private API key.
